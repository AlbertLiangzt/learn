static
	其中存放的是一些静态资源，如js、css等用于html的读取资源
template
	html存放文件，在demo.py文件中可以指定存放的路径，及初始加载的页面

pyweb原理
	一个用户请求过来先到了APP上，然后APP检查URLS，找到对应的模块，index，index检查是GET/POST请求就执行下面的代码，代码通过标准的模板渲染成网页，再返回给用户。

---------------------------------
demo.py
	用于启动web应用，将输入的内容进行获取，并且匹配输出，为之后的分词作准备

启动命令
	python demo.py 9999
访问路径
	http://192.168.224.10:9999/
